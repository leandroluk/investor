## Rules
- The system checks all registered `ConnectableContract` implementations.
- Returns `up` and uptime if all connections are healthy.
- Throws `UnhealthyError` (503 Service Unavailable) if any critical connection fails.

## Request
- **Method**: `GET`
- **Path**: `/support/health`

## Diagram

<MermaidZoom>
```mermaid
flowchart TD
    Start([Start]) --> Request[Client calls GET /support/health]
    Request --> PingDB[Ping Database]
    Request --> PingCache[Ping Cache]
    PingDB --> Check{All Healthy?}
    PingCache --> Check
    Check -- Yes --> Success[Return 200 OK]
    Check -- No --> Error[Return 503 Service Unavailable]
    Success --> End([End])
    Error --> End
```
</MermaidZoom>

## Success Case
- **Status**: `200 OK`
- **Body**:
```json
{
  "uptime": "1d 2h 30m"
}
```

## Error Case
- **Status**: `503 Service Unavailable`
- **Body**:
```json
{
  "code": "UNHEALTHY",
  "message": "Unhealthy"
}
```
